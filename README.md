# Azure Data Engineering Lab

Hands-on Azure data engineering exercises covering Azure Data Factory (ADF),
PySpark, and Azure Databricks.  
This repository is maintained as a learning-in-public workspace with clean,
interview-ready implementations.

---

## Repository Structure

adf/         → Azure Data Factory pipelines and ingestion patterns  
pyspark/     → PySpark transformation exercises  
databricks/  → Databricks-based Spark pipelines  
sql/         → Analytical and transformation queries  

Each folder contains multiple exercises, and each exercise includes its own
README describing the problem, approach, and outcome.

---

## Covered Topics

- Batch and incremental ingestion using Azure Data Factory
- Data transformations using PySpark
- Joins, aggregations, and window functions
- Medallion architecture (Bronze / Silver / Gold)
- Performance and optimization considerations

---

## Purpose

This repository focuses on practical data engineering patterns,
pipeline design, and transformation logic.
It intentionally avoids UI, dashboards, and production secrets.
